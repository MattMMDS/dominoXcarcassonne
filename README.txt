Indication du lancement du programme:

/!\ De préférence veuillez lancé le programme avec Java11. /!\

Une fois l'archive téléchargé;

1- Ouvrir le terminal de commande.
2- Se rendre dans le dossier groupe142 grace à la commande cd
3- Une fois situé dans le dossier , taper une des 3 commandes suivantes:
	
	1- javac Main.java
	2- java Main(#A vous de choisir un argument à mettre entre texte et graphique, vous pouvez également mettre 0 arguments#)
	Ou
	3- java -jar groupe142(#A vous de choisir un argument à mettre entre texte et graphique, vous pouvez également mettre 0 arguments#)

Le programme se lancera juste après cette commande.
